# EventPlanner

Sistema de gestión de eventos académicos desarrollado en Windows Forms
para mejorar la organización, inscripción y control de eventos dentro
del entorno SENA.

---

## Características principales

- Gestión de usuarios
- Creación y administración de eventos
- Inscripción de aprendices
- Generación de reportes
- Control de cupos y estados

---

## Tecnologías utilizadas

- C#
- .NET Framework 4.7.2
- Windows Forms
- SQL Server
- Git & GitHub

---

## Arquitectura

El proyecto sigue una arquitectura por capas:

Models → DAO → Services → Forms

---

## Instalación

1. Clonar repositorio
2. Ejecutar scripts en `/databases`
3. Abrir `EventPlanner.sln`
4. Ejecutar desde Visual Studio
