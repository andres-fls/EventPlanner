# Visión del Sistema — EventPlanner

## 1. Introducción

EventPlanner es una aplicación de escritorio desarrollada para apoyar la organización, gestión y control de eventos académicos dentro del entorno educativo del SENA. El sistema busca centralizar la información relacionada con eventos, participantes e inscripciones, reduciendo la desorganización generada por procesos manuales o dispersos.

---

## 2. Problema a Resolver

La gestión de eventos académicos suele realizarse mediante múltiples herramientas no integradas, registros manuales o información distribuida entre diferentes responsables. Esto genera problemas como:

- Pérdida o inconsistencia de información.
- Dificultad para realizar seguimiento de participantes.
- Falta de control sobre inscripciones y cupos.
- Complejidad para generar reportes o auditorías posteriores.
- Desorganización general en la planificación de eventos.

EventPlanner propone una solución centralizada que permite registrar y consultar toda la información del evento en un único sistema.

---

## 3. Objetivo del Sistema

Evitar la desorganización en la gestión de eventos mediante una plataforma que permita crear, administrar y controlar eventos académicos de forma estructurada y trazable.

---

## 4. Usuarios del Sistema

Actualmente el sistema contempla los siguientes tipos de usuario:

### Administrador
Responsable de:
- Crear y gestionar eventos.
- Administrar usuarios.
- Consultar información y reportes del sistema.
- Supervisar inscripciones.

### Aprendiz
Responsable de:
- Consultar eventos disponibles.
- Realizar inscripciones a eventos.
- Participar en actividades registradas dentro del sistema.

### Escalabilidad futura
El diseño del sistema permite incorporar nuevos roles en el futuro, tales como:
- Instructor
- Coordinador académico
- Otros perfiles administrativos.

---

## 5. Alcance del Sistema

El sistema está orientado principalmente al contexto institucional del **SENA**, aunque su arquitectura permite adaptaciones futuras para otros entornos educativos u organizacionales.

EventPlanner permite:

- Gestión de eventos.
- Control de inscripciones.
- Administración básica de usuarios.
- Registro estructurado de información para seguimiento y reportes.

---

## 6. Beneficios Esperados

- Centralización de la información.
- Mejor organización de eventos.
- Reducción de errores manuales.
- Mayor trazabilidad de actividades.
- Facilitar generación de reportes y auditorías.
- Base escalable para futuras mejoras.

---

## 7. Visión a Futuro

EventPlanner está diseñado bajo una estructura modular que permite evolucionar el sistema progresivamente, incorporando nuevas funcionalidades, roles y mejoras visuales sin afectar la estabilidad del núcleo del sistema.
